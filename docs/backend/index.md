# 后端

后端开发主要负责服务器端逻辑、数据库交互、API设计等核心功能。

## 主要技术栈

### Python
- **FastAPI**: 现代、高性能的 Web 框架
- **爬虫开发**: 数据采集与反爬虫技术
- **常用库**: pandas, numpy, requests 等

### Go
- **Gin**: 高性能的 Web 框架
- **并发编程**: goroutine 和 channel
- **微服务**: 适合构建分布式系统

### 数据库
- **MySQL**: 关系型数据库，企业级应用
- **SQLite**: 轻量级嵌入式数据库
- **Redis**: 高性能缓存数据库

## 学习路线

1. 掌握一门后端语言（Python/Go）
2. 学习数据库设计与操作
3. 理解 RESTful API 设计
4. 学习身份认证与授权
5. 部署与运维基础
